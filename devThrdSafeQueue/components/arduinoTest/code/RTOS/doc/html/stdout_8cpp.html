Detailed Description

stdout, the character stream used by the printf & co routines from the C standard library, is redirected into the stream Serial. Using printf, Arduino applications can communicate much easier with the console window as possible with the members of Serial for formatted writing. The idea of the code has been found in the Arduino Forum, at http://forum.arduino.cc/index.php?topic=120440.0, visited at June 12, 2013. It has been published by an anonymous author.

void init_stdout ( uint16_t  baudRateOfSerial)

Initialization: The redirection of stdout into Serial, mainly for use by printf & co, is done. This needs to be done prior to the first use of stdout and it may be done prior to the initialization of Serial.

Parameters
baudRateOfSerialThe stream Serial is initialized by this function. To do so, we need to know at which Baud rate it is run.
int puts_progmem ( const char *  string)

Write a null terminated string located in the CPU's flash ROM to stdout. End output with writing an additional newline character.

Returns
No failure is recognized and the function always returns the non-negative value 0.
Parameters
stringA pointer into the flash ROM.
Remarks
The function behaves like the function puts from the C library.
